**************************
 X-Match ZFOURGE & van der Wel et al. 2012 GALFIT catalogs
**************************

RELEASE: V0.4
DATE: August 8, 2016
SOURCE: http://zfourge.tamu.edu/
EMAIL: rebecca.allen85@gmail.com

**************************
         DETAILS
**************************

ZFOURGE v3.3 crossmatched with the GALFIT catalogs of van der Wel et. al 2014:

cdfs.v1.6.9  --------------  GOODS-S_F160W_galfit_v4.0.cat
cosmos.v1.3.6  ----------- COSMOS_F160W_galfit_v4.0.cat
uds.v1.5.8  --------------  UDS_F160W_galfit_v4.0.cat

**************************
        REVISIONS
**************************

v0.1 ---------------------  Initial release
v0.2 ---------------------  Formatting
v0.3 ---------------------  Formatting
v0.4 ---------------------  Catalogue updates

**************************
    CATALOGUE ENTRIES
**************************

id --------------------- ZFOURGE ID
wmin ------------------- wmin>0 for HST coverage
r  ---------------------- seperation of the ZFOURGE object ra and dec and the van der Wel et. al 2014 catalog ra and dec in arcseconds
f_r --------------------- flags the number of van der Wel et. al 2014 catalog matches to a single ZFOURGE object, f_r=0 for single matches, f_r=1 for multiple matches, f_r=2 for no matches
NUMBER ----------------- object id from van der Wel et. al 2014 catalogs
ra, dec ------------------ object ra and dec from van der Wel et. al 2014 catalogs
re, n, q, mag, and pa ------- structural parameters from van der Wel et. al 2014 catalogs, with errors dre, dn, dq, dmag,and dpa
sn ---------------------- object S/N from van der Wel et. al 2014 catalogs
f ----------------------- goodness of fit flag from van der Wel et. al 2014 catalogs

NOTES:

Sources from X_F160W_galfit_v4.0.cat were matched to ZFOURGE to within r<0.7”. If an object has multiple or no matches, the 'r' column has a value of -999.
