**************************
    CATALOGUE ENTRIES
**************************

id_zf  -------------------  ZFOURGE ID
id_3dhst  ----------------  3D-HST ID of nearest neighbour
use_3dhst  ---------------  3D-HST use flag
group_size  --------------  Number of 3D-HST sources within 0.7” of ZF source

NOTES:

3D-HST sources were matched to ZFOURGE to within r<0.7”. The group_size column indicates the number of sources matched within this radius. Only the ID of the nearest neighbour is displayed. 3D-HST data is available at http://3dhst.research.yale.edu/Home.html
